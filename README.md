# .github

This repository contains default community health files for repositories under **@gryfoleon**.

## What’s included

- `CODE_OF_CONDUCT.md`: collaboration standards
- `CONTRIBUTING.md`: contribution guidelines
- `SECURITY.md`: internal vulnerability reporting process
- `LICENSE`: Apache License 2.0
- `PULL_REQUEST_TEMPLATE.md`: PR template
- `ISSUE_TEMPLATE/bug_report.md`: bug report template
- `ISSUE_TEMPLATE/feature_request.md`: feature request template
- `ISSUE_TEMPLATE/task.md`: internal engineering task template
- `dependabot.yml`: Dependabot configuration
- `rulesets/`: GitHub repository rulesets

## How it works

GitHub may use these files as defaults for other repositories under this account when the target repository does not define its own copy.

If a repository contains its own `SECURITY.md`, `CONTRIBUTING.md`, etc., the repository-specific file takes precedence.

## Changes

Open a Pull Request with the proposed updates. Keep wording general so it applies across repositories.
